Apa itu HTML?

HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman web. Setiap halaman web yang Anda kunjungi dibangun menggunakan HTML. HTML terdiri dari serangkaian elemen (disebut juga tag) yang digunakan untuk mengelompokkan dan mengatur konten.

Apa itu CSS?

CSS adalah bahasa yang digunakan untuk mengatur tampilan dan gaya halaman web. Dengan CSS, Anda dapat mengontrol warna, font, tata letak, dan banyak aspek visual lainnya dari halaman web., atau dengan menghubungkan file CSS eksternal menggunakan tag link.

Menggabungkan HTML dan CSS

Untuk menggabungkan HTML dan CSS, Anda dapat menambahkan CSS langsung ke dalam file HTML menggunakan tag style di dalam head

Tag p

adalah salah satu elemen dasar dalam HTML yang digunakan untuk membuat paragraf teks.

Tag h1

adalah salah satu elemen HTML yang digunakan untuk membuat heading level 1 atau judul utama dalam sebuah halaman web.
